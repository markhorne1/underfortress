Underfortress — MVP

Quick start

1. Install dependencies:

```bash
cd /workspaces/underfortress
npm install
```

2. Start Expo:

```bash
npm start
```

This initial commit provides a minimal Expo + TypeScript skeleton, a small content loader, a `Home` screen and `Game` screen that loads the starting area and persists movement. Image prompts are preserved in the content JSON; missing images will show a prompt fallback in the dev panel.

Next steps: install deps, run the app, then I'll continue with engine/effects/combat and adding the larger sample content pack.
